Request Header Parser

API Basejump: Request Header Parser

Description:

By adding /api in the URL, you will recive a JSON contaning your IP address plus what language and operating system you are using. All information is derived from the request header via your browser.


Usage:

https://carlcal-header-parser-ms.herokuapp.com/api


Output Example:

{ "ipaddress": "72.198.91.159", "language": "en-US", "software": "Windows NT 6.1; WOW64" }
