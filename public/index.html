Request Header Parser

API Basejump: Request Header Parser

Description:

By adding /api in the URL, you will recive a JSON contaning the IP address, language and operating system you are using. All from from the request header via your browser. If an error occur when retriving the header. all the values in the JSON will be equal to null


Usage:

https://carlcal-header-parser-ms/api


Output:

{ "ipaddress": "xxx.xx.x.xxx", "language": "en-US", "software": "Windows NT 6.1; WOW64" }
